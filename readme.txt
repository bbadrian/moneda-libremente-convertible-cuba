== Description ==

Este plugin permite a los usuarios de WooCommerce agregar una nueva moneda llamada MLC (Moneda Libremente Convertible) a sus tiendas en línea. La MLC es una moneda utilizada comúnmente en los e-commerce en Cuba y su uso ha sido recientemente autorizado por el gobierno cubano.

Este plugin es fácil de instalar y configurar, y está diseñado para funcionar perfectamente con WooCommerce. Al agregar la MLC como una opción de moneda en tu tienda en línea, podrás aceptar pagos en esta moneda y expandir tus opciones de pago para los clientes en Cuba.

== Installation ==

Descarga el archivo zip del plugin a tu computadora.
En el escritorio de WordPress, navega a "Plugins" y haz clic en "Añadir nuevo".
Haz clic en "Subir plugin" y selecciona el archivo zip del plugin que descargaste.
Haz clic en "Instalar ahora" y espera a que se complete la instalación.
Activa el plugin.
Ve a la configuración de WooCommerce y agrega la MLC como una opción de moneda.

== Frequently Asked Questions ==

¿Este plugin es gratuito?
Sí, este plugin es completamente gratuito y está disponible para descargar en el repositorio de WordPress.

¿La MLC es una moneda oficial?
La MLC es una moneda autorizada recientemente por el gobierno cubano y es ampliamente utilizada en los e-commerce en Cuba.

¿Cómo puedo configurar el plugin para aceptar pagos en MLC?
Para aceptar pagos en MLC, simplemente agrega la MLC como una opción de moneda en la configuración de WooCommerce.
